# Local Demo Script

Run:

```powershell
C:\Anaconda\python.exe scripts/run_local_demo.py
```

The script prints:

1. initial factory state summary.
2. M3 overheat alert.
3. O4 urgent order.
4. task profile.
5. selected topology.
6. ReflAct agent trace.
7. infeasible-plan filters and violations.
8. final schedule.
9. alternative plans.
10. risk summary.
11. metrics.
12. Huawei adapter status.

It also writes:

- `reports/experiments/results.csv`
- `reports/experiments/summary.json`
- `reports/experiments/figures/README.md`
